# 艾新寵物診所 - 官方形象網站

## 🐾 介紹
艾新寵物診所官方形象網站，展示診所服務、團隊信息和寵物保健知識。

## 🛠️ 技術棧
- HTML5
- CSS3 (響應式設計)
- Vanilla JavaScript (無框架)
- GitHub Pages (部署)

## 🚀 本地開發設置

### 前提條件
- Node.js（非必需，純前端項目）
- Git
- VS Code
- Live Server 擴展（用於本地預覽）

### 開發步驟
```bash
# 1. Clone repository
git clone git@github.com:amostsai/catdog.git
cd catdog

# 2. 在 VS Code 中打開
code .

# 3. 使用 Live Server 預覽
# 右鍵 index.html → Open with Live Server
```

## 📦 部署到 GitHub Pages

1. 確保所有變更都已提交
2. 推送至 GitHub：
```bash
git push origin main
```

3. 在 GitHub Repository 設定中啟用 GitHub Pages：
   - Settings → Pages → Source: Deploy from a branch → main
   - 等待部署完成

4. 訪問：https://amostsai.github.io/catdog
